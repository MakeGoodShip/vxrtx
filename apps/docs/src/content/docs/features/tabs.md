## How It Works

vxrtx analyzes your open tabs and suggests logical groupings by topic, project, or activity — not just domain.

## Features

### Smart Grouping
AI examines tab titles (and URLs in YOLO mode) to create meaningful groups like "Work — Jira", "Research", or "Shopping".

### Duplicate Detection
Finds tabs with identical URLs and suggests closing the extras.

### Stale Tab Detection
Flags tabs you haven't visited in a while (configurable threshold in Settings).

### Full Preview Control
Every suggestion is a proposal. You can:
- Toggle groups on/off with checkboxes
- Rename groups by clicking the name
- Cycle colors by clicking the color dot
- Remove individual tabs from groups
- Undo after applying
