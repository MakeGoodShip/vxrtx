## OpenRouter (Recommended)

One API key, access to many models — Claude, GPT, Llama, Gemini, and more.

1. Get a key at [openrouter.ai](https://openrouter.ai)
2. Go to **Settings** > **AI Provider** > **OpenRouter**
3. Paste your key and select a model

### Popular Models
- **Claude Sonnet 4** — best quality
- **Claude Haiku 4** — fast and cheap
- **GPT-4o Mini** — fast and cheap
- **Llama 4 Scout** — free

## Direct API Access

You can also use Claude or OpenAI APIs directly:

- **Claude**: Get a key at [console.anthropic.com](https://console.anthropic.com)
- **OpenAI**: Get a key at [platform.openai.com](https://platform.openai.com)
