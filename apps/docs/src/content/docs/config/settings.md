## AI Privacy Tier
Choose how much data AI can access. See [Privacy Tiers](/features/privacy-tiers).

## AI Provider
Select OpenRouter, Claude, or OpenAI for cloud tiers. See [AI Providers](/config/ai-providers).

## Local AI Backend
When using the Secure tier, choose between Chrome Built-in AI and Ollama.

## Stale Tab Threshold
Number of days after which a tab is considered stale (default: 7). Tabs not accessed within this window will be flagged during organization.
